# neutron-chatt
Neutron chatt is a chatt application and tracking friends, build from React Native and Firebase(for Auth and Chatt DB using Firestore)

## Requirements
- Firesbase services like 
  - Firestore 
   - Auth Email/Password
   - Google Map Api Key
- React Native CLI
- NPM
- NodeJS
